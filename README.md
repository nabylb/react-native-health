# react-native-health

React Native abstraction wrapping Apple Healthkit and Google Fit to extract all available health data from apps, services, and wearables connected.

## Installation

```sh
npm install react-native-health
```

## Usage

```js
import Health from "react-native-health";

// ...

const result = await Health.multiply(3, 7);
```
